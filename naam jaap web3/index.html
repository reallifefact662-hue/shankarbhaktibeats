✨ Key Features

Name Input & Jaap Start

📿 Name Setup & Jaap Activation

📿 नाम सेट करें एवं जाप प्रारंभ

This feature allows the user to enter any name in the search bar (for example: Ram, Shiv, Krishna, or any custom name). Once the name is set, the user can start the chanting (Jaap) by tapping the button.

The Jaap can be started:

          🔊 With Voice (audio chanting enabled)

          🔇 Without Voice (silent counting mode)

This gives flexibility to both personal and public users..



इस फीचर में उपयोगकर्ता सर्च बार में कोई भी नाम लिख सकता है (जैसे: राम, शिव, कृष्ण या कोई भी इच्छित नाम)। नाम सेट करने के बाद उपयोगकर्ता टेप बटन दबाकर जाप शुरू कर सकता है।

जाप दो तरीकों से किया जा सकता है:

          🔊 आवाज़ के साथ (Voice Mode)

          🔇 बिना आवाज़ के (Silent Mode)

 यह सुविधा व्यक्तिगत और सामूहिक दोनों प्रकार के उपयोग के लिए उपयुक्त है।

Voice Speed Control
आवाज़ की गति नियंत्रण

🔊 Voice Speed Adjustment
🔊 आवाज़ की गति नियंत्रण

Users can control the chanting speed according to their comfort. Four speed options are available:


          ⚡ Fast – 0.5x
          🚀 Medium – 0.8x
          🎯 Normal – 1.0x
          🕊 Slow – 1.5x
After selecting the preferred speed, the user can start chanting using the tap button. This ensures a personalized spiritual experience..


उपयोगकर्ता अपनी सुविधा के अनुसार जाप की आवाज़ की गति चुन सकता है। इसमें चार विकल्प उपलब्ध हैं:


          ⚡ तेज – 0.5x
          🚀 मध्यम – 0.8x
          🎯 सामान्य – 1.0x
          🕊 धीमा – 1.5x
पसंदीदा गति चुनने के बाद, उपयोगकर्ता टेप बटन का उपयोग करके जाप शुरू कर सकता है। यह एक व्यक्तिगत आध्यात्मिक अनुभव सुनिश्चित करता है।

Voice Chanting On/Off
वॉइस ऑन / ऑफ नियंत्रण

⚡ Voice Toggle Control
⚡ वॉइस टॉगल सुविधा

This feature allows users to turn the chanting voice ON or OFF anytime.


        When ON → Audio chanting plays.
        When OFF → Only counting increases silently. This is useful for meditation, silent prayer, or group chanting sessions.


इस फीचर के माध्यम से उपयोगकर्ता कभी भी आवाज़ को चालू या बंद कर सकता है।


        ON करने पर → आवाज़ के साथ जाप होगा
          OFF करने पर → केवल काउंटिंग बढ़ेगी, बिना आवाज़ के
यह ध्यान, साधना या शांत वातावरण में उपयोग के लिए अत्यंत उपयोगी है।

Jaap Counter
जाप काउंटर

💾 Live Jaap Counting System
💾लाइव जाप काउंटिंग सिस्टम

Every time the user taps the button, the Jaap count increases and is displayed on the screen.


          Works in both voice and silent modes
          Shows real-time counting
          Helps users track their chanting progress accurately This ensures discipline and accurate repetition tracking.


हर बार टेप बटन दबाने पर जाप की संख्या स्क्रीन पर दिखाई देगी।


          आवाज़ और बिना आवाज़ दोनों मोड में कार्य करता है
          रियल-टाइम काउंटिंग दिखाता है
          उपयोगकर्ता अपनी प्रगति को सटीक रूप से ट्रैक कर सकता है
यह अनुशासन और सही गिनती बनाए रखने में मदद करता है।

Auto, Stop & Reset
ऑटो, स्टॉप एवं रीसेट

Automatic Chanting Control
लाइव जाप काउंटिंग सिस्टम


          🔁Auto Mode:Starts automatic chanting along with automatic count increase.
          ⏹ Stop Button: Stops both chanting and counting immediately.
          🔄 Reset Button: Resets the Jaap count to zero and allows the user to restart fresh. This feature provides complete control over the chanting session.


          🔁 ऑटो मोड:इस मोड में जाप और काउंटिंग स्वतः (Automatic) शुरू हो जाती है।
          ⏹ स्टॉप बटन:जाप और काउंटिंग को तुरंत रोक देता है।
          🔄 रीसेट बटन:सभी काउंटिंग को शून्य (0) पर रीसेट कर देता है, जिससे उपयोगकर्ता फिर से नया जाप शुरू कर सकता है।
यह फीचर उपयोगकर्ता को पूर्ण नियंत्रण प्रदान करता है।

Mala Count (Dot-Based Visual Counter)
माला काउंट (डॉट आधारित गिनती)

Digital Mala Counter
डिजिटल माला काउंटर

This feature displays counting in dot format, similar to traditional prayer beads (Mala).


          Default set to 1008 dots
          User can customize count: 11, 21, 51, 108, 1008, etc.
          The number entered in the search/count field updates the total dots
          When chanting starts, dots appear/activate according to the set count
This creates a traditional spiritual experience in digital form.


यह सुविधा पारंपरिक माला की तरह डॉट (बिंदु) के रूप में गिनती दिखाती है।


          डिफ़ॉल्ट रूप से 1008 डॉट्स सेट रहेंगे
          उपयोगकर्ता अपनी आवश्यकता अनुसार संख्या बदल सकता है: 11, 21, 51, 108, 1008 आदि
          जितनी संख्या सेट की जाएगी, उतने ही डॉट्स स्क्रीन पर दिखाई देंगे
         जाप शुरू होने पर डॉट्स क्रमशः सक्रिय होते जाएंगे
यह डिजिटल माध्यम में पारंपरिक आध्यात्मिक अनुभव प्रदान करता है।
